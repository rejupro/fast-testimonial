== Description ==
Fast Testimonial is a responsive WordPress Plugin.You can install it easily with a shortcode([fast-testimonial]). You can change testimonial author image, name, ratings also the feedback. This is fully customizable WordPress plugin where you can change your testimonial style dynamicly.
